# 2.3 产品测试效率对比

> **所属阶段**：第二阶段 - 负责人效能与产品洞察  
> **数据文件**：脱敏数据_特征增强.xlsx  
> **数据口径**：数据完全不去重，保留所有原始记录  
> **预计时间**：2-3小时

---

## 📊 分析目标

深入分析各子产品的测试效率和转化表现，识别高效产品与低效产品的差异特征，为产品策略优化和资源配置提供数据支撑。

**核心问题**：
- 各子产品的测试周期、转化率、流失率排名如何？
- 高效产品与低效产品在流失原因上有何差异？
- 哪些产品表现突出，哪些产品需要改进？

---

## 📋 需要获得的信息列表

### 1. 产品基础效能排名
- [ ] 各子产品的记录总数
- [ ] 各子产品的平均测试周期（天）、中位数、标准差
- [ ] **各子产品的申请→已接入转化率（%）**（核心指标）
- [ ] 各子产品的申请→已调用转化率（%）
- [ ] 各子产品的不接入流失率（%）

### 2. 产品不接入原因分布
- [ ] 各子产品的不接入原因分布（产品力不足/客户侧收益不足等）

### 3. 高效产品与低效产品对比
- [ ] 高效产品（转化率Top 2）的特征汇总
- [ ] 低效产品（转化率Bottom 2）的特征汇总

## ⚠️ 注意事项

1. **数据口径**：数据完全不去重，保留所有原始记录
2. **产品维度**：仅分析子产品，不分析售前主产品、系统主产品
3. **核心指标**：**申请→已接入转化率是核心指标**，已接入转化率 = 已接入数量 / 记录总数 × 100%
4. **流失率计算**：不接入流失率 = 不接入数量 / 记录总数 × 100%
5. **高效产品识别**：按转化率排序，取前2名作为高效产品
6. **输出路径**：所有文件保存到`执行分析结果/`文件夹
7. **数据文件**：使用特征增强数据文件（脱敏数据_特征增强.xlsx），直接使用预计算的状态标记和时间特征
