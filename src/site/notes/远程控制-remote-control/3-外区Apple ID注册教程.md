# 美区 Apple ID 注册教程

> 专门针对国区用户，从零开始注册美区 Apple ID 并下载 Tailscale。全程不需要信用卡，大约 10 分钟可完成。
>
> 这是 [2-远程控制-实操指南.md](./2-远程控制-实操指南.md) 的前置准备——国区 App Store 搜不到 Tailscale，需要美区账号。

---

## 第一部分：准备工作

### 你需要准备的东西

| 项目 | 说明 |
|------|------|
| **一个没用过的邮箱** | 建议新注册一个 Outlook / Gmail / 163 邮箱，不要用你已经绑定国区 ID 的那个 |
| **一部 iPhone** | 系统 iOS 13 以上即可 |
| **一个能收短信的手机号** | 用于账号验证（可以用你当前用的手机号，没问题） |
| **VPN 工具** | 后续登录 App Store 时需要切到美区节点 |

### 为什么要用美区？

Tailscale 不在国区 App Store 上架。美区账号下载一次后，**可以切换回国区账号**，App 不会消失。

---

## 第二部分：在网页上注册美区 Apple ID（推荐）

> **为什么不直接在手机上注册？** 手机注册会根据你的网络和设置自动锁定区域，很难绕过去。网页注册可以直接选美国，省去很多麻烦。

### 步骤 1：打开 Apple ID 注册页面

用手机自带的 **Safari 浏览器**（或电脑浏览器）打开：
```
https://appleid.apple.com/account
```

### 步骤 2：填写基本信息

| 字段 | 填写内容 |
|------|---------|
| **First Name（名）** | 随便写，比如 `San` |
| **Last Name（姓）** | 随便写，比如 `Zhang` |
| **Country/Region（国家/地区）** | 选 **United States**（美国） |
| **Date of Birth（生日）** | 随便填，确保大于 18 岁 |
| **Email（邮箱）** | 填你准备的新邮箱 |
| **Phone Number（手机号）** | 填你正在用的手机号（+86 中国手机号没问题，用于接收验证码） |

### 步骤 3：设置密码和安全问题

- **密码**：至少 8 个字符，包含大写字母、小写字母、数字
- **安全问题**：选三个问题随便填，**一定要记住答案**（建议截图保存）

### 步骤 4：验证邮箱和手机号

1. 填完后点击 **Continue**
2. 系统先给你的**邮箱**发验证码，抄下来填进去
3. 再给你的**手机号**发短信验证码，填进去
4. 完成后，你的美区 Apple ID 就创建好了

---

## 第三部分：在 iPhone 上登录美区 App Store

> **不要在"设置"里退出 iCloud 账号**，只退出 App Store 即可。这样照片、通讯录、备忘录都不会受影响。

### 步骤 1：退出当前 App Store 账号

1. 打开手机 **App Store**
2. 点击右上角的 **头像图标**
3. 滑到底部，点击 **"退出登录"**

### 步骤 2：登录美区账号

1. 再次点击头像图标
2. 输入你刚才注册的**美区 Apple ID** 和**密码**
3. 点击 **"登录"**

### 步骤 3：处理可能弹出的验证提示

第一次登录美区账号，可能会弹出：
> "此 Apple ID 尚未在 iTunes Store 使用过"

点击 **"检查"**（Review），在页面中：
- 确认国家是 **United States**
- **付款方式**：选 **None**（无）
- **账单地址**：如果需要填写，去 [meiguodizhi.com](https://www.meiguodizhi.com/) 生成一个免税州（俄勒冈、特拉华等）的地址填入即可

> 根据你的实际情况，账单地址页面可能不会出现，直接跳过了——遇到就填，没遇到就不管。

---

## 第四部分：打开 VPN 搜索并下载 Tailscale

### 步骤 1：开启 VPN

App Store 会检测你当前的 IP 归属地。即使登录了美区账号，如果 IP 是中国地址，可能还是搜不到 Tailscale。

打开你的 VPN，**选择美国节点**，确认 IP 已切到美国后再进 App Store。

### 步骤 2：搜索 Tailscale

1. 打开 App Store（已切 VPN）
2. 在搜索框输入 **Tailscale**
3. 这时应该能搜到了，图标是一个蓝色的 **T**

### 步骤 3：下载

- 点击 **"获取"** 下载
- 第一次下载要求**再次输入 Apple ID 密码**

### 步骤 4：换回国区账号（可选）

下载一次后，Tailscale 会一直留在手机上，即使换回国区账号也不会消失。

1. 打开 App Store → 点击头像 → 滑到底部 → 退出登录
2. 重新登录你的**国区 Apple ID**
3. Tailscale 依然在桌面上，可以正常使用

---

## 常见问题

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| **"This email is not available"** | 邮箱已被注册 | 换一个邮箱 |
| **收不到手机验证码** | 网络延迟 | 等 60 秒点击 Resend |
| **付款方式没有 None 选项** | 国家没选对 | 退出重新登录，确认国家是 United States |
| **登录美区后下载 App 要求绑定信用卡** | 下载了付费 App | Tailscale 是免费的，确认搜对了 |
| **搜不到 Tailscale** | IP 不在美国 | 开启 VPN 再试 |
| **以后怎么更新 Tailscale？** | — | 通常国区账号可以直接更新。如果提示无法更新，切回美区账号更新一下就行 |
